About the game

Numbra is a third-person stealth and exploration game developed in Unreal Engine 5 and published on Steam.

As a gameplay programmer, I focused on player movement, animation systems, and gameplay feel, working on systems that made character controls responsive and consistent throughout the game.

Gameplay Programming

  • Implemented core player mechanics, including movement and jumping
  • Developed responsive character controls
  • Connected gameplay logic and state transitions with the animation graph

Animation Graph

  • Integrated animations using animation state handling
  • Integrated blending spaces for smooth movement
  • Used animation notifies for more accurate event handling
  • Use root motion for some traversal mechanics (climbing)
  • Animation blending by bones

Collaboration and design

  • Collaborated with artists on skeleton setups and animation integration
  • Discussed additional animations for improving character feel
  • Contributed to gameplay decisions with the team
